# A Payroll Application

A simple java application to take employee wage inputs, calculate the gross pay, tax paid, bonus pay and net pay and create a payslip.

## Installation Steps

1. Clone the repository

```bash
git clone https://github.com/MazontheMoon/Payroll.git
```

2. Change the working directory

```bash
cd Payroll
```

3. Install dependencies

```bash
npm install
```

4. Run the app

```bash
npm run dev
```
